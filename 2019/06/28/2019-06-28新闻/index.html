决与择 不重生死，不论对错————《秦时明月-天行九歌》

新闻~~ 包含部分旧闻

1.国家主席习近平当地时间27日在大阪会见日本首相安倍晋三。习近平指出，当今世界正经历百年未有之大变局，全球治理体系深刻重塑，国际格局加速演变。中日两国拥有越来越多共同利益和共同关切。今年是中华人民共和国成立70周年，日本也进入令和时代。我们要共同致力于构建契合新时代要求的中日关系，使中日关系成为维护世界和平、促进共同发展的重要积极因素。

2.埃塞俄比亚媒体27日报道，中北部阿姆哈拉州首府巴赫达尔市22日发生未遂兵变以来，巴赫达尔和首都亚的斯亚贝巴将近250名嫌疑人被捕。

3.韩国统一部28日表示，政府通过联合国粮食计划署（WFP）对朝提供5万吨国产大米将用于帮助朝鲜境内的212万名弱势群体。

4.俄罗斯总统助理乌沙科夫当天向记者透露，普京在日本大阪与特朗普会晤期间邀请后者出席2020年俄罗斯举办的伟大卫国战争胜利75周年纪念活动。乌沙科夫说，后者对此予以积极回应。

5.伊朗常驻联合国代表马吉德·塔赫特·拉万齐26日说，除非英国、法国和德国采取及时和切实措施维护2015年达成的伊核协议，否则伊朗将突破该协议设定的低纯度浓缩铀存量限制。

6.一名波音官员表示，将于9月完成最新的737MAX系列客机软件更新。报道称，这意味着该系列客机至少到10月份才可能复飞。

7.美众院27日投票通过一项参院版的46亿美元紧急支出法案，为南部边境涌入的寻求庇护的儿童和家庭，提供亟需的人道主义援助。

8.欧洲部分国家这周迎来今夏首波强烈热浪，气温于这两天达到高峰。西班牙部分地区温度超过40摄氏度，已造成2人死亡；在法国，多达4000所小学关闭。

9.二十国集团领导人第十四次峰会28日在日本大阪举行。中国国家主席习近平出席并发表题为《携手共进，合力打造高质量世界经济》的重要讲话，强调二十国集团要坚持改革创新，挖掘增长动力；坚持与时俱进，完善全球治理；坚持迎难而上，破解发展瓶颈；坚持伙伴精神，妥善处理分歧。习近平宣布中国将进一步开放市场，努力实现高质量发展，为创造世界经济更加美好的明天不懈努力。
