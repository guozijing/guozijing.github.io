新闻~~ 包含部分旧闻

1.当地时间2019年6月21日，美国费城，费城最大炼油厂发生大规模爆炸并引起火灾，事件致5人受轻伤。据报道，事发炼油厂是拥有150年历史的费城能源解决方案有限公司（PES），爆炸也波及到了周边房屋。

2.国家主席习近平6月22日同爱尔兰总统希金斯互致贺电，庆祝两国建交40周年。

3.伊朗伊斯兰革命卫队空军指挥官阿米尔·阿里·哈吉扎德目前表示，如果华盛顿继续对德黑兰发动侵略，伊朗将扩大对“美国（军事）产品”的收藏。

4.　据美国《个人电脑》杂志6月22日报道，该杂志的一位英国记者被美国联邦快递公司告知，他不能将一部华为P30手机从英国快递到该杂志的纽约办公室。联邦快递在邮包标签上注明的理由是美国政府与华为、中国政府之间的一个问题，令其将这一手机送回发件人。

5.6月21日是国际瑜伽日，印度总理莫迪参加瑜伽表演活动。据悉，早些时候莫迪还通过社交媒体分享了以自己为原型的瑜伽卡通教学短片。

6.抗癌药入选首批鼓励仿制药建议清单 清单含34种药品 包括儿童白血病常用药巯嘌呤、艾滋病用药利匹韦林等 仿制药是指专利药品在专利保护期结束后，不拥有该专利的药企仿制的替代药品。仿制药与原研药具有相同的活性成分、剂型、给药途径和治疗作用。很多原研药价格昂贵，重症患者无力负担，价格便宜很多的仿制药成了他们的救命稻草。贫困患者迫切需要仿制药，但仿制药的供应和使用却遇到各种各样的障碍。

7.从一份美国政府于6月18日颁发、并于今天凌晨被多家美国媒体曝光的文件来看，被拉黑的5家中国企业和机构分别为芯片制造商海光、以及成都海光集成电路和成都海光微电子科技，超级计算机开发者中科曙光，以及无锡江南计算技术研究所（江南计算所）。

8.阿富汗警方22日说，由美国领导的北约驻阿联军21日夜间空袭位于坎大哈省的塔利班藏身处，击毙至少9名武装分子。

9.第34届东盟峰会22日至23日在曼谷举行，一系列东盟部长级会议也陆续登场。
本届东盟峰会以“推进可持续发展的伙伴关系”为主题。来自东盟各国的领导人将齐聚一堂，分享他们对可持续发展伙伴关系的愿景，并探讨加强东盟在防灾减灾、解决海洋垃圾以及文化交流等方面的合作。
