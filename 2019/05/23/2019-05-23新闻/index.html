新闻~~ 包含部分旧闻

1.2019年5月23日，我国时速600公里高速磁浮试验样车在青岛下线。这标志着我国在高速磁浮技术领域实现重大突破。

2.英国芯片设计公司ARM已经告诉员工，必须暂停与华为的业务往来。在文件中，ARM指示员工暂停与华为及其子公司的“所有有效合同、支持津贴，以及任何尚未签订的合约”，以遵守美国最近的一项贸易限制措施。

3.《泰晤士报》23日(当地时间)消息称，特雷莎⋅梅与英国内阁在“脱欧”问题上分歧不断。她预计会在24日(当地时间)宣布她将辞去首相一职的消息。

4.美国海军23日表示，美国舰艇与日本、澳大利亚及韩国等3国海军近日在西太平洋举行首次联合演习。此次演习是在美国总统特朗普本周末访问日本前夕举行的，代号为“太平洋先锋队”(Pacific Vanguard)，地点在关岛附近。据介绍，演习为期6天。包括2艘日本驱逐舰、2艘澳大利亚护卫舰以及1艘韩国驱逐舰参加。美国海军方面出动了5艘军舰，以及多架战斗机和海上巡逻机。演习内容包括实弹射击与反潜作战。

5.5月23日上午，南昌市公安局官方微博发布通报称：
5月15日17时许，南昌经开公安分局蛟桥派出所接到华东交通大学保卫处报案，称该校2015级学生张某于5月14日19时许离校后失联，至今未归。接报后，南昌经开公安分局立即成立工作组，会同南昌市公安局有关部门、蓝天救援队等单位对张某进行搜寻。5月22日上午10时40许，搜寻人员在赣江发现一具女性遗体。经张某家属辨认，女性遗体衣着与张某失联时穿着相似，体表没有明显外伤。目前，经DNA鉴定，已确认死者身份为失联学生张某，死亡原因正在调查中。

6.北京最新天气预报：26日或迎降雨，高温结束指日可待！

7.中国遥感33号卫星发射失败。

8.美东时间5月22日，美国国务院国际安全与防扩散局(Bureau of International Security and Nonproliferation)宣布对13个中国企业及个人实施制裁，其中包括友祥科技、浙江兆晨科技等10家企业以及3名个人，声明要求任何美国政府部门不得向这些企业或个人采购任何服务、产品、技术，不得与这些人员订立任何服务、产品、技术采购合同，不得向这些企业与人员提供任何协助，暂停授予这些人员现有的所有许可。
