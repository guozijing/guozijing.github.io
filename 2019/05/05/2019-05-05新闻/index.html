新闻~~ 包含部分旧闻

1、5月4日，以军轰炸了加沙地带约120个军事目标。 据加沙地带卫生部门当天发表的声明，以军当天对加沙地带的轰炸共造成4名巴勒斯坦人死亡、30名巴勒斯坦人受伤，死者中包括一名37岁孕妇和她年仅14个月大的女儿。

2、2019年伯克希尔股东大会如期在股神巴菲特家乡奥哈马举行，88岁的巴菲特和95岁的芒格再度联袂应对股东的车轮问答挑战，6个小时共回答了全球投资人55个问题。 戳这，全文

3、5月4日，美国佛罗里达州卡纳维拉尔角，SpaceX猎鹰9号火箭发射升空，搭载着CRS-17任务中的“龙”商业货运舱，从肯尼迪航天中心飞往国际空间站。“龙”号将向轨道实验室及其成员运送约5500磅的科学仪器、宇航员用品和硬件。

4、中国小伙在非洲当上部落酋长:中铁建中土尼日利亚有限公司运营事业部总经理孔涛，在尼日利亚获封部落酋长。

5、北京交大迎新校长 前校长因实验室爆炸事故被问责。5月5日，王稼琼任北京交通大学校长、党委副书记，宁滨不再担任。

6、朝公布发射导弹照片:朝鲜版”伊斯坎德尔”导弹亮相: 从朝鲜中央通讯社公布的照片来看，此次火力打击训练使用了被外界称为朝鲜版“伊斯坎德尔”的近程弹道导弹。

7、委内瑞拉军用直升机坠毁7人丧生: 正当委内瑞拉总统马杜罗在该国西部科赫德斯地区视察军事演习时，委军一架从首都加拉加斯飞往科赫德斯地区的直升机半途坠毁，机上七人全部丧生。

8、蒙古疑爆发鼠疫：2人死亡 100多人被隔离: 2名俄罗斯游客相继病死在蒙古。4日蒙古国家紧急事务管理局表示，这2人可能死于鼠疫，随后隔离了169人，其中11人被“无限期隔离”。

9、5月5日报道，据美联社报道，3日晚上美国伊利诺伊州的一家硅工厂发生爆炸，现已被证实有两人死亡。

10、火车“买短补长”临时办法下发：执意越站加收50%票款

11、铁幕徐徐落下！西方32国推5G安全准则拒华为: 来自欧盟、北约、美国、德国、日本和澳大利亚等西方32个国家和国家组织的代表，5月2－3日在捷克首都布拉格开了两天的5G安全准则讨论会，确定建立维护5G网络安全协调一致的政策措施。中国、俄罗斯等国没有受到会议邀请。虽然主办方声称并没有排斥哪个国家委派代表参会，但实际上这是西方国家的一个小圈子会议，或者说是一个应美国政府要求召开并且被美国政府主导的会议。
